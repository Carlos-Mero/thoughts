# Thoughts

Source code for my Bilibili channel "有所思".

「有所思，乃在大海南。」

大家好！这里是我新开设的栏目「有所思」，名称取自汉代诗歌铙歌十八曲当中的同名篇章，同时双关表达自己有所思考的含意。另一方面巧合的是它也刚好和英文当中 "thoughts" 一词谐音，我们的源代码仓库也就由此得名。之后这个系列会不定期更新，其中内容可能包括学界或业界的最新研究成果、新闻、我自己的评述、想法还有一些暴论。当然它们未必正确或者准确，我也希望和大家针对这些问题多做一些探讨。如果你感到其中的内容对你有所启发或者有所帮助的话，不妨给我点一个关注以及一键三连哦！

这里就是我的[Bilibili主页](https://space.bilibili.com/155818570)！此外markdown版本的文字稿也会在我的[知乎账号](https://www.zhihu.com/people/xiao-huang-ggg-97)下同步连载，欢迎关注！
